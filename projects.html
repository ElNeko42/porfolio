Proyectos

Pong

En este proyecto recreé el clásico juego de Pong utilizando HTML, CSS y JavaScript. El juego cuenta con una interfaz intuitiva y agradable a la vista, donde los jugadores pueden disfrutar de partidas rápidas y divertidas. Utilizando tecnologías web estándar, he logrado crear una experiencia de juego fluida y receptiva, llevando el juego retro de Pong al mundo moderno de la web.

Magic score

Desarrollé una aplicación integral para aficionados a Magic: The Gathering, utilizando Ionic y Angular, disponible tanto para Android como para iOS. Esta aplicación facilita el seguimiento de las partidas mediante contadores intuitivos y un marcador fácil de usar, mejorando la experiencia de juego. Aprovechando las capacidades multiplataforma de Ionic, la app ofrece una interfaz homogénea y amigable para usuarios en diferentes sistemas operativos, siendo una herramienta indispensable para cualquier aficionado a Magic.

Eshop eletronica

Mi proyecto eShop es una plataforma de comercio electrónico centrada en productos electrónicos, construida con un stack tecnológico robusto que incluye Laravel y Vue, complementado con Directus para una gestión de contenido ágil y eficiente. Esta tienda en línea no solo destaca por su interfaz amigable y diseño moderno, sino también por su back-end robusto que asegura una experiencia de compra segura y fluida para los usuarios. La integración con Directus permite una administración de productos y contenido sin fricciones, facilitando la expansión y escalamiento de la tienda con flexibilidad y eficiencia.

Juego de la botella

Elevé el clásico "juego de la botella" a un nuevo nivel con una aplicación desarrollada en Ionic, permitiendo a los usuarios disfrutar de este juego de fiesta popular directamente desde sus dispositivos móviles. La aplicación, disponible tanto para Android como para iOS, cuenta con una interfaz gráfica atractiva y jugabilidad sencilla, promoviendo la interacción y el entretenimiento entre amigos. Además, incluye características adicionales como personalización de tareas y modos de juego variados, añadiendo un giro moderno y emocionante al tradicional juego de la botella.

Pagina de limones

En este proyecto creativo, di vida a una plataforma de juego estilo "Cookie Clicker", pero con un giro cítrico: ¡limones! Desarrollada utilizando Vue.js, la web brinda una experiencia adictiva y estimulante donde los jugadores pueden cultivar su propio imperio de limones virtuales con cada clic. Con gráficos atractivos y mecánicas de juego progresivas, la página no sólo ofrece diversión sin fin, sino que también demuestra la flexibilidad y potencia de Vue para crear aplicaciones web dinámicas e interactivas.
